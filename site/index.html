How to Install

Get Hermes Agent running on your system in three simple steps.

1

Download

Click the download button for your platform above. The installer will download as a ZIP (macOS) or EXE (Windows).

2

Open Installer

Extract the ZIP on macOS and open "Hermes Installer.app", or run the EXE on Windows. You may need to approve the app in security settings.

3

Click Install

Choose your install location and click "Install Hermes". The installer will handle the rest and open the setup flow when complete.
